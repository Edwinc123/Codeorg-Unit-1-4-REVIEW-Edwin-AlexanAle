Unit 3

In unit 3 was focused on debugging, pair programming, user interface, and event driven programs. Some new things that we learned was input, which is data that is sent to a computer for processing by a program. They can come in different forms like tactile interaction, audio,visuals, or text. Outputs which is data that is sent from a program to a device, they can come in different forms like inputs. And user interfaces that use inputs and outputs that allow a user to interact with a piece of software and can include a variety of forms like buttons,menus,images,text and graphics.

Variables are placeholder/containers that are used to store a single piece of data whether its numbers,numbers, etc and variables keep code organized and easier to read .
